A **lightweight browser-based HTML editor** that allows users to write and preview HTML instantly. The project is designed to make coding simple, fast, and accessible directly from the browser.

---

## Live Editor

Launch the editor here:

**https://goateditor.vercel.app/**

No downloads or installations are required.

---

## Features

* Live HTML editing
* Instant preview of code
* Clean and simple interface
* Runs completely in the browser
* Fast and lightweight
* CSS editor support
* JavaScript editor support
* Multiple file support
* Project saving

---

## Quick Start

1. Open the editor website.
2. Type HTML code in the editor window.
3. Click **Run**.
4. View the result in the preview panel.

Example code to try:

```html
<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>

<h1>Hello World</h1>
<p>This page was created using the HTML Editor.</p>

</body>
</html>
```

## Goals of the Project

The project is built around a few key ideas:

**Simplicity**
Users should be able to open the editor and start coding immediately.

**Accessibility**
The editor works directly in the browser, meaning it can run on many devices.

**Fast Feedback**
Users can quickly see how their HTML code changes affect the page.

**Learning**
The tool is designed to help beginners experiment and learn web development.
